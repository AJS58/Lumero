# Lumero v0.2 Interactive Prototype

This is the first interactive GitHub Pages version of Lumero.

## What works in v0.2
- Branded splash screen
- Tabbed dashboard interface
- Smart scan animation
- Local image upload testing
- Basic sharpness/blur scoring using browser canvas analysis
- Best-shot recommendation from uploaded images
- Simple report generation
- Emergency recovery simulation

## Important
This is still a browser prototype, not a finished iOS App Store app. Real iPhone Photos integration requires the native SwiftUI build using Apple's Photos framework, Vision and CoreML.
