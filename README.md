# Lumero v0.1 Fixed Full Build

This package includes both:

1. A GitHub Pages web preview via `index.html`.
2. The native SwiftUI starter files inside the `Lumero` folder.

## How to upload to GitHub Pages

Upload **all files and folders** in this package to the root of your `Lumero` GitHub repository.

Your repository root should contain:

- `index.html`
- `README.md`
- `VERSION_NOTES.md`
- `lumero-branding-concept.png`
- `Lumero/` folder

Then visit:

`https://ajs58.github.io/Lumero/`

## Important

The previous version opened as a text instruction page because there was no proper `index.html` at the repository root. This fixed full version solves that.

## Native iOS app

The SwiftUI files are still included for Xcode development. The web preview is only a visual/prototype landing page and is not the native app itself.
