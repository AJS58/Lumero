# Lumero v0.6 Full Working Web PWA

Upload the contents of this folder directly to the root of the GitHub `Lumero` repository.

Important:
- Do not upload the folder itself.
- The repo root should contain `index.html`, `manifest.json`, `service-worker.js`, `version.json`, `reset.html`, `.nojekyll`, and `assets/`.

After upload:
1. Open `https://ajs58.github.io/Lumero/reset.html`
2. Tap the reset button.
3. Then open `https://ajs58.github.io/Lumero/index.html?v=0.6`
4. Delete and re-add the Home Screen icon if the old version appears.

This is still a web/PWA prototype, not a native iOS app.
