# Lumero v0.1 with Branding Assets

This package contains the GitHub Pages web preview plus the Lumero icon/logo/splash assets.

## Upload to GitHub Pages
Upload the contents of this folder to the root of the `Lumero` repository. The root should contain `index.html`, `.nojekyll`, `favicon.png`, `assets/`, and `LumeroApp/`.

## Included
- App icon PNG sizes
- Xcode `AppIcon.appiconset`
- Lumero logo PNG and SVG
- iPhone splash screens
- GitHub Pages preview
- SwiftUI starter files
